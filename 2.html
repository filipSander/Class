Что такое CMS

CMS - это «Система управления контентом» (Движок) – написанная PHP-программистом основа
для сайта, с помощью которой вы можите управлять сайтом (добовлять контент, менять пункты
меню и т.п.) не зная HTML и CSS

Однако, для того чтобы сделать сайт с помощью CMS потребуются услуги и программиста, и
дизайнера, и верстальшика. И капиталовложения.

Какие бывают CMS

Бывают различные системы управления контентом: для интернет-магазинов, для блогов, для
форумов и т.д.

Примеры CMS

Примеры популярных CMS: Joomla, WordPress (для блогов), PhpBB (для форумов).

CSM-ки бывают платные и бесплатные.
